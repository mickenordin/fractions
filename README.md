**Compile and run**
```bash
git clone https://github.com/mickenordin/fractions.git
cd fractions
g++ -o fractions fractions.cpp
./fractions
```

**Output**

```
a = 3+2i
a* = 3-2i
a*a* = 13
b = 4-3i
b* = 4+3i
b*b* = 25
a + b = 7-i
(a + b)* = 7+i
a - b =  -1+5i
(a - b)* = -1-5i
a * b =  18-i
(a * b)* = 18+i
a / b = (6/25)+(17/25)i
(a / b)* = (6/25)-(17/25)i
```
